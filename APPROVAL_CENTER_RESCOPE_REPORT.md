# Approval Center Rescope Report

Approval Center is now scoped as enterprise/governance diagnostics, not the default Gmail onboarding controller.

Sections:

- Provider Coverage
- Auto Approved Gmail
- Google OAuth Blocked
- OAuth Success
- Enterprise Pending
- Approved Requests
- Rejected Requests
- Expired Requests

Default Gmail auto-approved rows do not appear as admin-pending. `oauth_success` maps to OAuth Success, never Pending. `oauth_failed` maps to Google OAuth Blocked.
